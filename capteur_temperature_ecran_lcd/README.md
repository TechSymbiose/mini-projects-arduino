# Temperature sensor

Project done by Adrien Louvrier, 07/01/2021
The goal is to display the tempereture measure on a LCD display
card used : Mega 2560

# Electronic circuit
input : 
	- DHT11 temperature sensor (DHTPIN -> 53, Vcc -> 5V, grd -> grd)

output : 
	- LCD display (rs -> 2, en -> 3, d4 -> 4, d5 -> 5, d6 -> 6, d7 -> 7)
